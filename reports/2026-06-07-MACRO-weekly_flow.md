|---------|---------|
| 非农+17.2万 → 市场定价加息概率52% → 2年期收益率+7bp至4.05% → 风险资产估值全面承压 |
| 加息预期 → 实际利率预期上行 → 黄金-3.09%（避险资产同样承压） |
| 加息预期 → 美元走强预期 → 新兴市场资金外逃 → KOSPI-7.14% |
| 加息预期 → 科技股（高估值/长久期资产）最脆弱 → 纳斯达克-5.07% |

### ⚡ 核心结论

> **本周是2025年4月"解放日"关税暴跌以来最惨烈的一周。但最值得关注的不是科技股跌了多少，而是——黄金也跌了。**
>
> **"现金为王"是本周唯一正确的交易。** 非农超预期+AI去泡沫形成"双杀"：加息预期压制所有资产的估值（包括黄金），AI泡沫刺破打击风险偏好。结果是**股、债、金、油、币全面下跌**——没有任何资产能提供正回报。

> **黄金-3.09%是本周最重要的信号。** 它告诉我们：当利率预期急剧转向时，黄金的避险属性会被"持有成本上升"完全压制。这与2022年Fed激进加息时黄金下跌的逻辑一致。
>
> **KOSPI-7.14%是最值得警惕的信号。** 韩国股市作为"全球风险偏好的金丝雀"，其暴跌幅度远大于其他市场，可能预示着更深层的全球科技股去泡沫才刚刚开始。
>
> **下周关键：6月10日5月CPI数据。** 若CPI超预期→更深层"现金为王"（黄金可能继续跌，科技继续承压）；若CPI回落→报复性反弹可期（黄金和科技可能同步反弹）。
